## 概要
戦闘ロボット「リボルバーシックス」を操作し、敵を倒していくシンプルで少しハードな2.5D横スクロール
アクション。

残弾を管理しつつ、ジャンプ・ダッシュ・攻撃を駆使してステージを踏破せよ。

## 操作方法

*振動はPS5コントローラーに最適化しています
| 操作 | キーボード & マウス | パッド (XInput / PS) |
|---|---|---|
| スタート | Enter  | Start |
| 移動 | WASD / 矢印キー  | 左スティック / Dパッド |
| ジャンプ | Spaceキー  | A / X |
| ショット | E / 右クリック | X / □ |
| (長押しでチャージ) |
| クイックショット | 攻撃が敵に当たった後再ショット |
| リロード | R / ↓↓ | B / 〇 / ↓↓ |
| ダッシュ | Shift | LB / L1 |
| 攻撃モードチェンジ | Q | RB / R1 |
